# Listas

Las listas se definen mediante números (ordenadas) o mediantes los símbolos `*` o `-` (desordenadas).

::: customDemo

```markdown
### Listas ordenadas

1. Wake up at 6
2. Eat an apple
3. Go for walk
4. Take shower
5. Time to work

### Listas desordenadas

- Extended markdown
- Themes for support
  - Default
  - Elegant
  - Paper
  - Mono
- In built linter
```
